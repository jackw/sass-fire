    omit

    @function omit($keys, $map) { ... }

    Description

    Returns a partial copy of an object omitting the specified keys

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $keys

    keys to remove from the map

    		List— none
    $map

    the map to copy from

    		Map— none

    Returns

    Map —

    A new map without the properties from $keys.

    Example

    omit

    $map-omit: omit((tom, harry), (tom: 1, dick: 2, harry: 3);
// $map-omit -> (dick: 2);

    Description

    Returns a single item by iterating through the list, successively calling the iterator function and passing it an accumulator value and the current value from the array, and then passing the result to the next call.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $fn

    		Function— none
    $acc

    The accumulator value.

    $list

    		Array— none

    Returns

    Any type —

    The final, accumulated value.
